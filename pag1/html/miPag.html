Encabezados de una página Web

Los encabezados de una página web utilizan la etiqueta < h# >, donde # es un número que va del 1 al 6; 1 para el encabezado más grande y 6 para el más pequeño.

Etiqueta DIV

Para alinear la imágen de Google al centro, se utilizó la etiqueta div, la cual sirve para definir una sección dentro del documento HTML. Esta etiqueta normalmete se usa para agrupar un bloque de elementos, para luego añadirle un estilo determinado.
